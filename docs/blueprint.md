# **App Name**: MeritBase

## Core Features:

- Freelancer Onboarding: Light KYC process for freelancers involving email, phone, and government ID upload.
- Employer Onboarding: Company license upload for verification.
- CVWallet Creation: Auto-generated profile from user input with editable fields (skills, experience, portfolio).
- Decentralized Storage: Storing CV data and other associated documents on IPFS, including cryptographic hash stored on the blockchain.
- Job Posting: Employers create job posts with detailed requirements, using the description field, and the jobs are persisted to the smart contract on the blockchain.
- AI-Powered Job Matching Tool: Matching engine calculates the fit percentage between the job description and the freelancer's CVWallet, using a HuggingFace sentence similarity model to act as a tool and determine degree of matching.
- Reputation System: Employers leave reviews & badges on task completion, with data updates stored immutably via blockchain transactions to CVWallet after each job. Employers can browse previous job histories for all freelancers.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) to evoke trust and professionalism in the decentralized gig platform.
- Background color: Light gray (#E0E0E0) to offer a clean and neutral backdrop.
- Accent color: Lime green (#32CD32) for highlighting important actions, giving a sense of growth and opportunity.
- Font pairing: 'Space Grotesk' sans-serif for headers, 'Inter' sans-serif for body.  'Space Grotesk' gives a modern, computerized feel for headlines; use 'Inter' for longer text.
- Use simple, professional icons from Phosphor or Tabler to maintain clarity.
- Clean and structured layouts for both freelancer and employer dashboards, optimized for easy navigation.
- Subtle transitions and loading animations to provide a smooth user experience without distracting from the main content.
